# Touch Up Solutions Rebuild

A recreated and modernized e-commerce-style website for Touch Up Solutions.

## Overview

This project rebuilds the Touch Up Solutions website with a cleaner interface, structured product pages, repair categories, how-to resources, and a more polished shopping experience.

## Features

- Home, product, resources, programs, and video pages
- Industrial shop sections
- Wood, leather, metal, and accessory repair categories
- Product data handled through JavaScript
- Checkout page mockup
- Responsive static website structure

## Tech Stack

- HTML
- CSS
- JavaScript
- Vercel
